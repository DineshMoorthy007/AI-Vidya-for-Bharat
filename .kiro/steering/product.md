# AI Vidya for Bharat - Product Overview

AI Vidya for Bharat is an AI-powered, multilingual learning platform democratizing education across India through vernacular content generation.

## Core Mission
Bridge the urban-rural digital divide by providing structured courses, localized content, and interactive assessments in 12+ Indian languages.

## Target Users
- Students in Tier 2/3 cities preferring native language learning
- Skill-seekers and farmers needing vocational/agricultural training in regional dialects
- Educators requiring quick curriculum generation in multiple languages

## Key Differentiators
- Voice-first input using Bhashini/IndicWhisper for accessibility
- Context-aware video integration with language-filtered YouTube content
- Code-switching support (Hinglish, Tanglish) for natural learning
- Offline/low-data mode for rural connectivity challenges
- Mobile-first design optimized for Indian network conditions

## Success Metrics
- 60%+ courses generated in non-English languages
- High completion rates for generated courses
- Strong voice command adoption for course creation
